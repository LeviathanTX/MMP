# Methane Mitigation Partners

Website for Methane Mitigation Partners — converting idle wells into verified climate assets.

## Overview

Methane Mitigation Partners permanently plugs end-of-life oil and gas wells and converts the eliminated emissions into independently verified, insured carbon credits.

## Local Development

Simply open `index.html` in a browser to view the site locally.

## Deployment

This site is a static HTML page and can be deployed to any static hosting service (GitHub Pages, Netlify, Vercel, etc.).
